전주공업고등학교 졸업생 네트워크 플랫폼 완벽 활용 가이드

🎯 학생 계정 개요

학생 계정은 재학생과 졸업생이 사용하는 계정으로, 채용 정보 조회, 취업박람회·산업체견학·자격증지원 신청, 진로 상담, 동문 네트워킹, 경력 관리 등의 기능을 제공합니다.

📝 1. 회원가입 및 로그인

회원가입 방법

  1. 메인 페이지에서 "회원가입" 버튼 클릭
  2. 필수 정보 입력:
    • 이름
    • 이메일 (로그인 ID로 사용)
    • 비밀번호 (6자 이상)
    • 전화번호
    • 학교 선택
    • 사용자 유형 (재학생/졸업생/교사)
    • 졸업년도 · 전공 (재학생/졸업생만)
  3. "가입하기" 버튼 클릭 → 서버 DB에 저장됨
  4. 성공 메시지 확인 후 로그인 페이지로 자동 이동
💡 안내: 회원가입 정보는 서버에 저장되므로 다른 기기에서도 동일한 계정으로 로그인할 수 있습니다.

로그인 방법

  1. 등록한 이메일과 비밀번호 입력
  2. "로그인" 버튼 클릭
  3. 대시보드로 자동 이동
💡 Mac 사용자 안내: macOS에서 이메일이 자동으로 대문자로 변환되는 경우 로그인이 실패할 수 있습니다. 이메일을 직접 소문자로 입력해주세요.

📊 2. 대시보드 활용하기

로그인 후 첫 화면인 대시보드에서 모든 주요 정보를 한눈에 확인할 수 있습니다.

통계 카드

  • 📊 신규 채용공고: 최근 등록된 채용공고 수
  • 📝 지원한 공고: 내가 지원한 채용공고 수
  • 🤝 네트워크: 연결된 동문 수
  • 💬 메시지: 받은 메시지 수

추천 채용공고

최신 채용공고를 미리보기로 확인할 수 있습니다.

최근 소식

관리자가 등록한 공지사항을 확인할 수 있습니다. 항목 클릭 시 상세 내용 팝업으로 표시됩니다.

평생교육프로그램

등록된 교육 프로그램 목록을 확인할 수 있습니다. 항목 클릭 시 기간, 마감일, 수강료, 설명을 확인할 수 있습니다.

💼 3. 채용 정보 활용하기

채용공고 검색 및 필터링

  1. 좌측 메뉴에서 "채용 정보" 클릭
  2. 키워드 검색 (직무명, 회사명), 지역, 근무형태, 경력 등 필터 사용

채용공고 지원

  1. 공고 상세 페이지에서 "지원하기" 버튼 클릭
  2. 지원 양식 작성 후 제출

🎪 4. 취업박람회

지역 기업들이 참여하는 취업박람회 정보를 확인하고 참가 신청을 할 수 있습니다.

취업박람회 목록 확인

  1. 좌측 메뉴에서 "취업박람회" 클릭
  2. 등록된 박람회 목록 확인 (주최기관, 일정, 장소, 참가비 등)
  3. 필터로 원하는 박람회 검색

참가 신청

  1. 원하는 박람회 카드의 "참가 신청" 버튼 클릭
  2. 신청 정보 입력 (이름, 연락처, 이메일, 메시지)
  3. "신청 확인" 버튼 클릭
💡 안내: 로그인 상태에서는 이름과 이메일이 자동으로 입력됩니다.

🏭 5. 산업체 견학

지역 기업 현장 견학 프로그램 정보를 확인하고 신청할 수 있습니다.

산업체 견학 목록 확인

  1. 좌측 메뉴에서 "산업체 견학" 클릭
  2. 등록된 견학 프로그램 목록 확인 (기업명, 일정, 모집인원, 혜택 등)

견학 신청

  1. 원하는 견학 프로그램 카드의 "견학 신청" 버튼 클릭
  2. 신청 정보 입력 후 "신청 확인" 클릭

📜 6. 자격증 지원

자격증 취득을 위한 지원 프로그램 정보를 확인하고 신청할 수 있습니다.

자격증 지원 목록 확인

  1. 좌측 메뉴에서 "자격증 지원" 클릭
  2. 등록된 지원 프로그램 목록 확인 (자격증명, 지원내용, 마감일 등)
  3. 유형별 필터 (기사, 기능사, IT자격증 등)로 검색 가능

지원 신청

  1. 원하는 프로그램 카드의 "지원 신청" 버튼 클릭
  2. 신청 정보 입력 후 "신청 확인" 클릭

💬 7. 진로 상담 신청하기

상담 신청 방법

  1. 좌측 메뉴에서 "진로 상담" 클릭
  2. "상담 신청하기" 버튼 클릭
  3. 상담 유형 선택 (진로, 취업, 학업, 기타), 제목, 내용 입력
  4. "신청하기" 버튼 클릭

상담 내역 확인

  • 🟡 대기중: 교사 승인 대기
  • 🟢 승인됨: 상담 승인 완료 (교사 답변 확인 가능)
  • 🔴 거절됨: 상담 거절 (사유 확인)

🤝 8. 동문 네트워킹

동문 검색

  1. 좌측 메뉴에서 "동문 네트워킹" 클릭
  2. 이름, 졸업년도, 재직 회사, 직무 분야 등으로 검색

네트워크 연결 및 메시지

  1. 동문 프로필에서 "네트워크 추가" 버튼 클릭
  2. 연결된 동문에게 "메시지 보내기" 가능

👤 9. 내 프로필 및 경력 관리

프로필 수정

  1. 좌측 메뉴에서 "내 프로필" 클릭 → "프로필 수정"
  2. 전화번호, 자기소개, 관심 분야, SNS 링크 등 수정 가능

경력 · 자격증 · 교육 · 포트폴리오 관리

  1. 좌측 메뉴에서 "경력 관리" 클릭
  2. 탭 선택: 경력 / 자격증 / 교육 / 포트폴리오
  3. 각 탭에서 추가 버튼으로 정보 입력 후 저장

🎓 교사 계정 개요

교사 계정은 학생들의 진로 상담을 관리하고, 기업을 대신하여 채용공고를 등록·관리할 수 있는 계정입니다.

📊 1. 교사 대시보드

주요 섹션

  • 내 채용공고: 내가 등록한 채용공고 목록
  • 최근 지원자: 최근 지원한 학생 목록
  • 상담 신청 내역: 학생들의 진로 상담 요청
  • 최근 소식 / 평생교육프로그램: 관리자 등록 공지 확인

💼 2. 채용공고 관리

채용공고 등록

  1. 대시보드에서 "공고 등록" 버튼 클릭
  2. 회사명, 직무명, 근무지역, 급여, 고용형태, 경력/학력 요구, 직무설명, 자격요건, 복리후생, 마감일 입력
  3. "등록하기" 버튼 클릭

채용공고 수정/삭제

  1. 내 채용공고 목록에서 해당 공고 선택
  2. "수정" 또는 "삭제" 버튼 클릭
⚠️ 주의: 채용공고를 삭제하면 해당 공고의 지원자 정보도 함께 삭제됩니다.

👥 3. 지원자 관리

  1. 대시보드의 "최근 지원자" 섹션에서 지원자 목록 확인
  2. 지원자 이름 클릭 → 학력, 경력, 자격증, 포트폴리오 등 상세 프로필 확인
  3. 평가 및 메모 작성 가능

💬 4. 진로 상담 관리

상담 확인 및 답변

  1. 좌측 메뉴에서 "진로 상담" 클릭
  2. 대기 중인 상담 요청 클릭
  3. "승인" (답변 작성) 또는 "거절" (사유 작성) 선택
  4. "저장" 버튼 클릭

상담 상태

  • 🟡 대기중: 처리 전
  • 🟢 승인됨: 답변 완료
  • 🔴 거절됨: 거절 처리

⚙️ 관리자 계정 개요

관리자는 시스템의 모든 데이터를 관리하고 모니터링할 수 있는 최고 권한을 가진 계정입니다.

관리자 전용 메뉴
  • 👥 회원 관리
  • 💼 채용공고 관리
  • 📋 게시판 관리 (최근 소식 · 평생교육프로그램)
  • 📣 공지사항 관리 (취업박람회 · 산업체견학 · 자격증지원)
  • ⚙️ 코드 관리 (학교 · 전공)

👥 1. 회원 관리

회원 목록 조회

  1. 좌측 메뉴에서 "회원 관리" 클릭
  2. 유형별 필터 (전체 / 학생 / 졸업생 / 교사 / 기업 / 관리자) 및 이름·이메일 검색

회원 정보 수정

  1. 회원 목록에서 "수정" 버튼 클릭
  2. 이름, 이메일, 전화번호, 사용자 유형, 학교, 졸업년도 수정 가능
  3. "저장" 버튼 클릭

회원 삭제

  1. 회원 목록에서 "삭제" 버튼 클릭 → 확인
⚠️ 주의: 회원을 삭제하면 해당 회원의 모든 데이터(상담 내역, 지원 정보 등)가 함께 삭제됩니다.

💼 2. 채용공고 관리

  1. 좌측 메뉴에서 "채용공고 관리" 클릭
  2. 상태별 필터: 전체 / 활성 / 마감 / 만료
  3. "공고 등록" 버튼으로 신규 공고 추가
  4. 공고 선택 후 "수정" 또는 "삭제" 처리
  5. 마감일 지난 공고는 "만료" 처리하면 학생 목록에서 제외됨

📋 3. 게시판 관리

최근 소식 관리

  1. 좌측 메뉴에서 "게시판 관리" 클릭
  2. "최근 소식" 탭 선택 → "소식 등록" 버튼
  3. 제목, 내용 입력 후 "등록" 클릭
  4. 수정/삭제는 목록에서 해당 버튼 클릭

평생교육프로그램 관리

  1. "평생교육프로그램" 탭 클릭 → "프로그램 등록"
  2. 프로그램명, 설명, 시작일, 종료일, 신청 마감일, 수강료 입력
  3. "등록" 클릭
💡 팁: 등록된 최근 소식과 교육프로그램은 모든 사용자의 대시보드에 자동으로 표시됩니다.

📣 4. 공지사항 관리 (취업박람회 · 산업체견학 · 자격증지원)

학생들이 조회·신청할 수 있는 취업박람회, 산업체 견학, 자격증 지원 정보를 등록·관리합니다.

공지사항 등록

  1. 대시보드에서 "공지사항 관리" 클릭 (또는 admin-announcements.html 접속)
  2. "새 공지사항 등록" 버튼 클릭
  3. 공지 정보 입력:
    • 유형: 취업박람회 / 산업체견학 / 자격증지원
    • 제목: 행사명
    • 주최기관: 주관 기관명
    • 행사일: 행사 날짜
    • 행사시간: 운영 시간
    • 장소: 행사 장소
    • 신청 마감일: 신청 마감 날짜
    • 모집 인원: 최대 참가 인원
    • 참가비: 무료 또는 금액
    • 혜택: 제공 혜택 (쉼표로 구분)
    • 참가 조건: 자격 요건 (쉼표로 구분)
    • 연락처 / 이메일: 문의 정보
    • 태그: 검색 태그 (쉼표로 구분)
    • 이미지 URL: 대표 이미지 링크 (선택사항)
    • 상세 URL: 외부 상세 페이지 링크 (선택사항)
  4. "저장" 버튼 클릭

공지사항 수정

  1. 공지사항 목록에서 수정할 항목의 "수정" 버튼 클릭
  2. 내용 변경 후 "저장" 클릭

공지사항 삭제

  1. 목록에서 "삭제" 버튼 클릭 → 확인
💡 안내: 등록된 공지사항은 학생 메뉴의 취업박람회/산업체견학/자격증지원 페이지에 즉시 반영됩니다.
⚠️ 주의: 공지사항을 삭제하면 해당 공지의 신청 내역도 함께 삭제됩니다.

⚙️ 5. 코드 관리

회원가입 시 선택 항목으로 사용되는 학교, 전공 정보를 관리합니다.

학교 추가/수정/삭제

  1. 좌측 메뉴에서 "코드 관리" 클릭
  2. "학교 추가": 학교명 입력 후 저장 → 회원가입 학교 선택 항목에 자동 추가
  3. 목록에서 "수정" / "삭제" 버튼으로 관리
⚠️ 주의: 해당 학교로 등록된 회원이 있는 경우 먼저 회원의 학교 정보를 변경해야 합니다.

📊 6. 데이터 모니터링

관리자 대시보드에서 전체 사용자 수, 활성 채용공고 수, 진행 중인 상담 건수, 최근 등록된 소식 및 교육 프로그램을 실시간으로 확인할 수 있습니다.

💡 팁 및 주의사항

보안 관련

  • 비밀번호는 6자 이상 사용
  • 공용 컴퓨터 사용 후 반드시 로그아웃
  • 이메일은 소문자로 입력 (Mac 사용자 주의)

데이터 관리

  • 정기적으로 프로필 정보 업데이트
  • 만료된 채용공고는 정기적으로 정리
  • 중요한 상담 내역은 별도로 기록 보관

브라우저 호환성

  • Chrome, Edge, Firefox 최신 버전 권장
  • Internet Explorer는 지원하지 않음

🔧 문제 해결

로그인 안 될 때

  1. 이메일과 비밀번호를 정확히 입력했는지 확인
  2. 브라우저 쿠키 및 캐시 삭제 후 재시도
  3. 다른 브라우저(Chrome, Edge)로 시도

Mac(맥)에서 로그인 안 될 때

  2. 이메일을 직접 소문자로 다시 입력하세요.

데이터가 보이지 않을 때

  2. 브라우저 캐시 강제 삭제 (Ctrl+Shift+R)
  3. 인터넷 연결 상태 확인
  4. 다른 브라우저로 시도

회원가입 후 로그인이 안 될 때

  1. 가입 시 사용한 이메일과 비밀번호 확인

## 시스템 개요

전주공업고등학교 졸업생 네트워크 플랫폼은 재학생, 졸업생, 교사, 기업이 함께 소통하고 취업 정보를 공유할 수 있는 통합 플랫폼입니다.

### 주요 기능
- 👤 **프로필 관리**: 개인 정보 및 경력 관리
- 💼 **채용 정보**: 기업 채용공고 조회 및 지원
- 🤝 **동문 네트워킹**: 졸업생 및 재학생 간 네트워크 형성
- 💬 **진로 상담**: 교사와 학생 간 1:1 진로 상담
- 📈 **경력 관리**: 경력, 자격증, 교육 이력, 포트폴리오 관리

시스템 테스트를 위한 계정 정보입니다.

- **이메일**: `student@test.com`
- **비밀번호**: `student123`
- **이름**: 김학생
- **권한**: 채용정보 조회, 진로상담 신청, 동문 네트워킹

### 교사 계정
- **이메일**: `teacher@test.com`
- **비밀번호**: `teacher123`
- **이름**: 이선생
- **권한**: 채용공고 등록, 상담 승인/거절, 지원자 관리

### 관리자 계정
- **이메일**: `admin@admin.com`
- **비밀번호**: `admin123`
- **이름**: 관리자
- **권한**: 전체 시스템 관리 (회원, 채용공고, 게시판, 코드 관리)

## 학생 계정 매뉴얼

### 1. 회원가입 및 로그인

#### 회원가입
1. 메인 페이지에서 **"회원가입"** 버튼 클릭
2. 필수 정보 입력:
   - 이름
   - 이메일 (본인 인증용)
   - 비밀번호 (8자 이상)
   - 전화번호
   - 학교 선택 (관리자가 등록한 학교 목록에서 선택)
   - 사용자 유형 선택 (재학생/졸업생)
   - 졸업년도 (졸업생만)
3. **"가입하기"** 버튼 클릭
4. 회원가입 완료 메시지 확인

#### 로그인
1. 등록한 이메일과 비밀번호 입력
2. **"로그인"** 버튼 클릭
3. 대시보드로 자동 이동

### 2. 대시보드 사용하기

로그인 후 첫 화면인 대시보드에서 다음 정보를 확인할 수 있습니다:

#### 통계 카드
- 📊 **신규 채용공고**: 최근 등록된 채용공고 수
- 📝 **지원한 공고**: 내가 지원한 채용공고 수
- 🤝 **네트워크**: 연결된 동문 수
- 💬 **메시지**: 받은 메시지 수

#### 추천 채용공고
- 최신 채용공고 3개 미리보기
- **"지원하기"** 버튼: 채용정보 페이지로 이동
- **"자세히보기"** 버튼: 상세 정보 확인

#### 최근 소식
- 관리자가 등록한 학교 공지사항
- 클릭하면 상세 내용 확인 가능

#### 평생교육프로그램
- 등록된 교육 프로그램 목록
- 프로그램 클릭 시 상세 정보 (기간, 수강료, 내용) 확인

### 3. 채용 정보 활용하기

#### 채용공고 검색
1. 좌측 메뉴에서 **"채용 정보"** 클릭
2. 검색 필터 사용:
   - 키워드 검색 (직무명, 회사명)
   - 지역별 필터
   - 근무형태 (정규직, 계약직, 인턴 등)
   - 경력 요구사항

#### 채용공고 상세보기
1. 관심 있는 공고 클릭
2. 상세 정보 확인:
   - 회사명 및 위치
   - 직무 설명
   - 자격 요건
   - 우대 사항
   - 급여 및 복리후생
   - 마감일

#### 채용공고 지원
1. 공고 상세 페이지에서 **"지원하기"** 버튼 클릭
2. 지원 양식 작성 (자동으로 프로필 정보 불러오기)
3. 제출 확인

### 4. 진로 상담 신청하기

#### 상담 신청
1. 좌측 메뉴에서 **"진로 상담"** 클릭
2. **"상담 신청하기"** 버튼 클릭
3. 상담 정보 입력:
   - 상담 유형 선택 (진로, 취업, 학업, 기타)
   - 제목 입력
   - 상담 내용 상세 작성
4. **"신청하기"** 버튼 클릭

#### 상담 내역 확인
1. 진로 상담 페이지에서 **"내 상담 내역"** 탭 클릭
2. 상담 상태 확인:
   - 🟡 **대기중**: 교사 승인 대기
   - 🟢 **승인됨**: 상담 승인 완료 (교사 답변 확인)
   - 🔴 **거절됨**: 상담 거절 (사유 확인)

#### 상담 답변 확인
1. 승인된 상담 항목 클릭
2. 교사의 답변 및 조언 확인
3. 추가 질문이 있는 경우 메시지 기능 사용

### 5. 동문 네트워킹

#### 동문 검색
1. 좌측 메뉴에서 **"동문 네트워킹"** 클릭
2. 검색 필터 사용:
   - 이름 검색
   - 졸업년도별 필터
   - 재직 중인 회사
   - 직무 분야

#### 네트워크 연결
1. 연결하고 싶은 동문 프로필 클릭
2. **"네트워크 추가"** 버튼 클릭
3. 연결 요청 메시지 작성 (선택)
4. 상대방 승인 대기

#### 메시지 보내기
1. 연결된 동문 목록에서 선택
2. **"메시지 보내기"** 버튼 클릭
3. 메시지 작성 및 전송

### 6. 내 프로필 관리

#### 기본 정보 수정
1. 우측 상단 사용자 이름 클릭 또는 좌측 메뉴에서 **"내 프로필"** 클릭
2. **"프로필 수정"** 버튼 클릭
3. 수정 가능 항목:
   - 프로필 사진
   - 전화번호
   - 자기소개
   - 관심 분야
   - SNS 링크 (GitHub, LinkedIn 등)
4. **"저장"** 버튼 클릭

### 7. 경력 관리

#### 경력 사항 등록
1. 좌측 메뉴에서 **"경력 관리"** 클릭
2. **"경력"** 탭에서 **"경력 추가"** 버튼 클릭
3. 경력 정보 입력:
   - 회사명
   - 직위
   - 근무 기간
   - 담당 업무
4. **"저장"** 버튼 클릭

#### 자격증 등록
1. 경력 관리 페이지에서 **"자격증"** 탭 클릭
2. **"자격증 추가"** 버튼 클릭
3. 자격증 정보 입력:
   - 자격증명
   - 발급 기관
   - 취득일
   - 유효기간 (선택)
4. **"저장"** 버튼 클릭

#### 교육 이력 등록
1. **"교육"** 탭에서 **"교육 추가"** 버튼 클릭
2. 교육 정보 입력:
   - 교육명
   - 교육 기관
   - 교육 기간
   - 교육 내용
4. **"저장"** 버튼 클릭

#### 포트폴리오 등록
1. **"포트폴리오"** 탭에서 **"포트폴리오 추가"** 버튼 클릭
2. 포트폴리오 정보 입력:
   - 프로젝트명
   - 프로젝트 설명
   - 참여 기간
   - 사용 기술
   - 프로젝트 링크 (GitHub, 웹사이트 등)
3. **"저장"** 버튼 클릭

## 교사 계정 매뉴얼

### 1. 로그인 및 대시보드

교사 계정으로 로그인하면 기업 계정과 유사한 대시보드가 표시됩니다.

#### 대시보드 구성
- 📝 **활성 채용공고**: 등록한 채용공고 수
- 👥 **총 지원자**: 채용공고에 지원한 학생 수
- 👁️ **조회수**: 채용공고 조회 수
- ✅ **채용 완료**: 채용이 완료된 건수
- 💬 **상담 신청 내역**: 학생들의 진로 상담 요청 목록

### 2. 채용공고 관리

교사는 기업을 대신하여 채용공고를 등록하고 관리할 수 있습니다.

#### 채용공고 등록
1. 대시보드에서 **"공고 등록"** 버튼 클릭
2. 채용 정보 입력:
   - **회사명**: 채용 기업명
   - **직무명**: 채용 포지션
   - **근무 지역**: 근무지
   - **급여**: 급여 정보
   - **고용 형태**: 정규직, 계약직, 인턴 등
   - **경력 요구**: 신입/경력 및 년수
   - **학력 요구**: 최소 학력 요건
   - **직무 설명**: 상세 업무 내용
   - **자격 요건**: 필수 자격 요건
   - **우대 사항**: 우대 조건
   - **복리후생**: 복지 혜택
   - **마감일**: 지원 마감일
3. **"등록하기"** 버튼 클릭
4. 등록 완료 메시지 확인 후 대시보드로 이동

#### 채용공고 수정
1. 대시보드의 "내 채용공고" 섹션에서 수정할 공고 선택
2. **"수정"** 버튼 클릭
3. 정보 수정 후 **"저장"** 버튼 클릭

#### 채용공고 삭제
1. 수정 화면에서 **"삭제"** 버튼 클릭
2. 삭제 확인 후 완료

### 3. 지원자 관리

#### 지원자 목록 확인
1. 대시보드의 "최근 지원자" 섹션 확인
2. 각 지원자의 기본 정보 확인:
   - 이름
   - 지원 직무
   - 지원일
   - 상태 (검토중, 합격, 불합격)

#### 지원자 상세 정보 확인
1. 지원자 이름 클릭
2. 지원자 프로필 확인:
   - 학력 정보
   - 경력 사항
   - 자격증
   - 포트폴리오
3. 평가 및 메모 작성

### 4. 진로 상담 관리

#### 상담 신청 확인
1. 대시보드의 "상담 신청 내역" 섹션 확인
2. 또는 좌측 메뉴에서 **"진로 상담"** 클릭
3. 대기 중인 상담 요청 목록 확인

#### 상담 승인 및 답변
1. 상담 요청 항목 클릭
2. 학생의 상담 내용 확인
3. 두 가지 옵션 선택:
   - **"승인"**: 상담 승인 후 답변 작성
   - **"거절"**: 거절 사유 작성
4. 답변 또는 거절 사유 입력
5. **"저장"** 버튼 클릭

#### 상담 내역 관리
1. 진로 상담 페이지에서 상태별 필터:
   - 🟡 대기중
   - 🟢 승인됨
   - 🔴 거절됨
2. 각 상담 건 클릭하여 추가 답변 또는 수정

### 5. 최근 소식 및 교육프로그램 확인

교사 대시보드에서도 학교의 최근 소식과 평생교육프로그램을 확인할 수 있습니다.
- 항목 클릭 시 상세 정보 확인
- 학생들에게 공유 가능

## 관리자 계정 매뉴얼

관리자는 시스템의 모든 데이터를 관리하고 모니터링할 수 있는 최고 권한을 가집니다.

### 1. 관리자 대시보드

#### 대시보드 통계
- 🤝 **네트워크 수**: 전체 사용자 수
- 💼 **지원한 공고**: 전체 채용공고 수
- 📝 **저장한 공고**: 상담 신청 수

#### 관리자 메뉴
좌측 사이드바에서 다음 메뉴에 접근할 수 있습니다:
- 👥 **회원 관리**
- 💼 **채용공고 관리**
- 📋 **게시판 관리**
- ⚙️ **코드 관리**

### 2. 회원 관리

#### 회원 목록 조회
1. 좌측 메뉴에서 **"회원 관리"** 클릭
2. 전체 회원 목록 확인
3. 필터 사용:
   - **유형별**: 전체, 학생, 교사, 졸업생, 기업, 관리자
   - **검색**: 이름 또는 이메일로 검색

#### 회원 정보 수정
1. 회원 목록에서 **"수정"** 버튼 클릭
2. 수정 가능 항목:
   - 이름
   - 이메일
   - 전화번호
   - 사용자 유형
   - 학교
   - 졸업년도
3. **"저장"** 버튼 클릭

#### 회원 삭제
1. 회원 목록에서 **"삭제"** 버튼 클릭
2. 삭제 확인 팝업에서 **"확인"** 클릭
3. 회원 데이터가 완전히 삭제됨

#### 회원 추가
1. 상단의 **"회원 추가"** 버튼 클릭
2. 신규 회원 정보 입력
3. **"저장"** 버튼 클릭

### 3. 채용공고 관리

#### 채용공고 목록 조회
1. 좌측 메뉴에서 **"채용공고 관리"** 클릭
2. 전체 채용공고 목록 확인
3. 상태별 필터:
   - 전체
   - 활성 (모집중)
   - 마감
   - 만료

#### 채용공고 등록
1. 상단의 **"공고 등록"** 버튼 클릭
2. 채용 정보 입력 (교사 매뉴얼의 채용공고 등록 참조)
3. **"등록하기"** 버튼 클릭

#### 채용공고 수정/삭제
1. 채용공고 목록에서 해당 공고 선택
2. **"수정"** 또는 **"삭제"** 버튼 클릭
3. 수정 시 정보 변경 후 저장
4. 삭제 시 확인 후 완전 삭제

#### 채용공고 만료 처리
1. 마감일이 지난 공고의 **"만료"** 버튼 클릭
2. 공고 상태가 "만료"로 변경됨
3. 더 이상 학생들에게 표시되지 않음

### 4. 게시판 관리

게시판 관리에서는 학교 공지사항과 교육 프로그램을 관리합니다.

#### 최근 소식 관리

**소식 등록**
1. 좌측 메뉴에서 **"게시판 관리"** 클릭
2. **"최근 소식"** 탭 선택 (기본값)
3. **"소식 등록"** 버튼 클릭
4. 소식 정보 입력:
   - 제목
   - 내용
5. **"등록"** 버튼 클릭

**소식 수정**
1. 소식 목록에서 **"수정"** 버튼 클릭
2. 제목 또는 내용 수정
3. **"저장"** 버튼 클릭

**소식 삭제**
1. 소식 목록에서 **"삭제"** 버튼 클릭
2. 삭제 확인 후 완료

#### 평생교육프로그램 관리

**프로그램 등록**
1. **"평생교육프로그램"** 탭 클릭
2. **"프로그램 등록"** 버튼 클릭
3. 프로그램 정보 입력:
   - 프로그램명
   - 프로그램 설명
   - 시작일
   - 종료일
   - 신청 마감일
   - 수강료 (무료인 경우 "무료" 입력)
4. **"등록"** 버튼 클릭

**프로그램 수정/삭제**
1. 프로그램 목록에서 **"수정"** 또는 **"삭제"** 버튼 클릭
2. 수정 시 정보 변경 후 저장
3. 삭제 시 확인 후 완료

### 5. 코드 관리

코드 관리에서는 시스템에서 사용되는 기준 정보를 관리합니다.

#### 학교 관리

**학교 등록**
1. 좌측 메뉴에서 **"코드 관리"** 클릭
2. **"학교 관리"** 섹션에서 **"학교 추가"** 버튼 클릭
3. 학교명 입력
4. **"저장"** 버튼 클릭
5. 등록된 학교는 회원가입 시 학교 선택 항목에 자동으로 추가됨

**학교 수정**
1. 학교 목록에서 **"수정"** 버튼 클릭
2. 학교명 변경
3. **"저장"** 버튼 클릭

**학교 삭제**
1. 학교 목록에서 **"삭제"** 버튼 클릭
2. 삭제 확인 팝업에서 **"확인"** 클릭
3. ⚠️ 주의: 해당 학교로 등록된 회원이 있는 경우 먼저 회원의 학교 정보를 변경해야 함

#### 기본 학교 정보
- 시스템 최초 실행 시 "전주공업고등학교"가 기본으로 등록됨
- 필요에 따라 추가 학교 등록 가능
